================================================
  부동산 실거래가 조회 앱 사용법
================================================

[ 최초 1회 (설치) ]
  1. 이 폴더 안의 [ setup_최초1회.bat ] 파일을 더블클릭
  2. 검은 명령창에서 자동으로 Streamlit 등을 설치합니다
     (1~3분 소요, 인터넷 필요)
  3. "설치 완료!" 메시지가 나오면 창을 닫아도 됩니다

[ 평소 사용 ]
  1. [ 부동산앱_시작.bat ] 더블클릭
  2. 검은 창이 잠깐 뜨고, 자동으로 브라우저(크롬/엣지)가 열립니다
     주소: http://localhost:8501
  3. 좌측 사이드바에서
       · 시도 → 시군구 선택
       · 시작월 ~ 종료월 선택
       · (선택) 키워드, 평형, 가격대 필터
     설정 후 [조회] 버튼 클릭
  4. 결과는 화면에 표로 표시
     · [Excel 다운로드] : 단지별 요약 + 전체 거래내역 2시트 xlsx
     · [HTML 다운로드] : 단일 .html 파일 (오프라인 보관용)
  5. 종료할 때는 검은 창에서 Ctrl+C 또는 창 닫기

[ 자주 묻는 질문 ]

Q. 브라우저가 안 열려요.
A. 수동으로 크롬을 열고 주소창에 http://localhost:8501 입력

Q. "MOLIT_API_KEY 없음" 오류
A. C:\Users\ajfxo\.claude\.env 파일에 키가 있는지 확인
   파일 내용 예: MOLIT_API_KEY=발급받은긴문자열

Q. "조회 실패" 오류
A. 국토부 API 일일 한도(보통 1만 건) 초과 가능성.
   다음 날 자정 이후 재시도. 또는 한 번에 시군구 1~2개만.

Q. 화면이 이상하거나 안 뜸
A. 검은 창 한 번 닫고 [부동산앱_시작.bat] 다시 더블클릭

Q. 데이터는 어디서 오나요?
A. 국토교통부 공공데이터포털 (실거래가 공개시스템 동일 데이터).
   계약일 기준 신고분이며, 최근 1~2개월은 누적 중일 수 있습니다.
